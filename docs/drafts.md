  - 掌握获取信息的主动权，而不是被信息支配

- [ ] 信息质量的定义与评判标准 #card

  - 对于不同的信息获取目标，可能会有不同的评判标准。
  - 首先，"高质量" 的定义是什么？
    - 世界充满了矛盾的信息，没有识别能力就会左右摇摆，我们应该如何才能保持自我？
    - 从客观的角度出发
      - 煽动或利用情绪的信息，一般属于劣质信息
      - 缺乏事实支撑的信息，凭空的观点陈述，一般属于劣质信息；
      - 相反，包含事实支撑，作者长期且具体实践的信息，属于优质信息
    - 从个人的角度出发
      - 让个人过于舒适的信息，一般对个人而言属于劣质信息；（信息茧房）
        - 可以通过多问一句“<mark>这条信息让我获得了什么新知或启发？</mark>”来判断。
  - 错过的信息或者被遗忘的信息，都是不重要的信息。
    - 人们通常害怕错过信息，哪怕他们知道98%的信息可能是无用的。这对知识类工具的设计可能很重要，也对ChatGPT这类AI应用的功能有所启示。我们可以使用ChatGPT帮助我们筛选信息，不过需要具体分析ChatGPT在阅读网页时是否会错过重要信息，以及这种情况发生的可能性和原因。
